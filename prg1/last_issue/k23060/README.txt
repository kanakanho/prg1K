行列計算をして戦うマルバツゲーム

ルール
基本的にはマルバツゲームと同じ  
プレイヤーは交互にマスを指定、あるいは行列の計算をして、縦横斜のいずれか 1 行揃えることを目指す

入力と操作
プレイヤーは O（先行） と X（後攻）の 2 人

まず最初に計算記号を指定する
「+」の場合
1〜9 の中からマークの入力場所を選ぶ  
なお、他のマークで埋まっている場合は入力できない

「x」の選択
1〜9 の順番に「0」か「1」を入力していく（それ以外の数字は入力できない）  
1 は最大で 3 つまで、各行、各列に 1 つづつ  
0 は最大で 8 つまで、全てを 0 で埋めることはできない

入力された値の通りに計算を行うことでできること
- 任意の行の入れ替え
- 任意の行の 0 倍
- 任意の行を 0 倍して入れ替え

特徴
- 行列計算を行うことで入力後もマークの移動や変更が可能
- 通常のマルバツゲームのように全てのマスが埋まっても終わらない
- ゲームで勝利するためには行列計算の知識が必要
